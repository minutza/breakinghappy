Breaking Happy

how to use github (deutsch)

  1. Terminal öffnen
    1. ls (=lists all folders)
    2. cd Desktop
    3. ls
    4. cd WEB
    5. ls
    6. cd breakinghappy [tab drücken]
  2. git status
    1. git add . (sagt was gespeichert(commited) werden soll)
    2. git commit -m "Nachricht" (speichert mit Nachricht)
  3. git checkout -b branch (erstellt und wechselt in die branch)
  4. git branch - d/ - D branch (löscht die branch)
  5. git branch (zeigt in welcher branch ich mich befinde und welche es noch gibt)
  6. git checkout branch (wechselt in einer anderen branch)
  7. git merge branch (speichert von branch in master)
  8. git push (speichert von master in origin/master)
  9. git pull (for up-to-date)
